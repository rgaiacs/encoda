decodeObject

  • decodeObject(ext: Extension): object
    • Defined in codecs/md/index.ts:1046

    Decode an !object inline extension to an Object.

    Valid objects include (the first is the preferred and the default, the last should be avoided):

    • !object("key":value, ...) (comma separated pairs, values can be any JSON primitives)
    • !object{key=string ...} (space separated pairs; values can only be strings)
    • !object (decoded to {})
    • !object["key":"value", ...]

    • ext: Extension

    Returns object
